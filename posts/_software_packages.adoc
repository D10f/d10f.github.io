== Debian Packages

[source,bash]
----
$ apt list --installed
$ dpkg-query --show --showformat='${Package}\t${Version}\t${Installed-Size}\n'   | column -tN Package,Version,Installed-Size   | sort -nk3
----

== Flatpak

[source,bash]
----
$ flatpak list
$ flatpak --columns=app,name,size,installation
----
